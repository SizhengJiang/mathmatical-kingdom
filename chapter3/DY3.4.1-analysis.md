这个文件里主要讲述的是我思考一致连续的过程，陈老在讲述一致连续性时我当时不懂的地方以及国王模型的诞生。

首先先说明一致连续性的概念：设函数f(x)在区间X上定义，若对任意给定的ε>0，存在δ>0，只要x',x"∈X，满足丨x′-x″丨<δ，就成立丨f(x′)-f(x″)丨<ε，则称函数f(x)在区间X上一致连续。

当时陈老在讲一致连续性这个概念时就说过这是数学分析的第一个门槛，第一座大山，我在初接触这个概念时我也很懵。
因为陈老在讲这块的时候联动了很多之前的知识，我也是翻看前面的笔记然后再反复听陈老的视频，对陈老的说的每一句话都进行反复的推敲与斟酌我才弄懂，但这也只是开胃小菜而已。
因为我推敲的也只是陈老对一致连续性概念或者定义的一种字面解释，陈老在说完δ(ε)能不能找到 它跟函数本身有关也跟X区间有关时，当然到这以前我都能理解，但紧接着陈老就抛出了一个重磅炸弹：
陈老就假设了δ(ε)>0存在，就让你推它的充要条件是什么。陈老接下来的这些话令我陷入了沉思。

陈老说：“令所有适用的δ(x0,ε)中最大者（最大者不一定有，那就上确界）为δ*(x0,ε)，那么δ(ε)存在的充要条件是什么？其实就是δ*(x0,ε)当中最小的那个大于0。”

陈老又补充了一句，这句话也是我领悟国王模型至关重要的一句话：“给一个x0有一个最大的δ*(x0,ε)，给一个x0有一个最大的δ*(x0,ε)。” 所以“我”要在所有的δ*（x0,ε）当中取下确界，但如果取了下确界它就有可能等于0；所以如果“我”要想让δ(ε)>0存在那就必须在后面填个>0的条件，即：存在δ(ε)>0 ⇔ inf x0∈X δ*(x0,ε)>0。

在听第一遍的时候脑子都炸了，什么玩意，陈老怎么就得出答案了，什么就对吧，行吧，这都什么跟什么啊。在只看最后结果的情况下，怎么就冒出个上确界的下确界，这都什么跟什么啊！
后来当我冷静下来听第二遍时，依旧没听出来什么端倪。

于是我索性把陈老说过的话一字不落地记下来，在纸上反复揣摩斟酌。
结果还是不行，因为实在是太抽象了。陈老并没有对此做出过多的解释，只是简单的把最后的结果告诉你。

我始终不能理解：为什么要在δ(x0,ε)中取最大者（上确界），最后又要下确界？
后来我尝试先顺着陈老的思路先选择追溯本源，然后再尝试用比喻的方法进行理解。

真要追溯本源其实陈老的思路，核心思想无外乎就是俩字：假设。

我们要倒着想，为什么偏偏选最大者上确界呢？其实就是要想我们到底想干什么，来看看我们的已知条件。

我们已知：函数f在区间X上每一点都连续。这意味着：对任意给定的ε>0，以及对每一个固定的点x0∈X，都存在一个依赖于该点的δ(x0,ε)>0，使得连续性的定义成立。
那现在我们想问：什么情况下，f在X上是一致连续的？

一致连续性要求：对给定的ε>0，能找到一个统一的，不依赖于点的δ(ε)>0，使得定义对“所有” “点对”同时成立，所以陈老做出了假设：

我们能不能用那些个体户δ(x0,ε)来拼凑，集所有人之长来托出一个万能的δ(ε)呢？

那回到我们的问题，咋就非得选最大者呢？

当时我想过很多比喻，但后来想着想着发现都不太合适，但到后来也不知道怎么回事就鬼使神差的想到了一部动漫和一句话：我附庸的附庸不是我的附庸。

受此启发，我构建了一个“国王模型”，把数学符号一一对应成故事里的角色。
| 比喻 | 数学对应 | 角色说明 |
|:---|:---:|---:|
| 国王 | 点x0 | 任选，谁当国王都行，选了就不能动了 |
| 权杖 | 半径δ | 划分疆域的能力，由国情（函数）和法规（ε）主动寻找确定 |
| 王法标准值 | f(x0) | 国王不能改变自己的函数值 |
| 法律容忍度 | 任意给定的ε>0 | 外部给定的，国王不能抱怨法律太严 |
| 守法子民范围 | {x: |f(x)-f(x0)|<ε } | 国王无法改变这个事实范围 |
| 国王的实际疆域 | 即安全半径（x0-δ,x0+δ） |   国王实际统治的范围   |
| 合法性要求 | (x0-δ，x0+δ)‌⊆‌丨f(x)-f(x0)丨<ε | 疆域内所有人都必须守法 |

在此处再插一句话：想到这里我终于明白了，其实我们以前在证由|x-x0|<δ ⇒ |f(x)-f(x0)|时，我们知道<ε连续，>ε则不连续，其实本质上就是在证
(x0-δ，x0+δ‌)⊆‌丨f(x)-f(x0)丨<ε

那我们回到刚才的问题，对于给定的一个ε，同一个ε，不同点x0需要的“安全半径”,而δ(x0,ε)大小不一。

举个例子y=1/x,对于同一个ε，不同位置的国王需要的“安全半径”δ(x0,ε)可能相差极大。例如：

国王在x0=100处：函数平缓，可用δ=10
国王在x0=0.01处：函数陡峭，只敢用δ=0.0001

一个天真的想法：如果我取所有δ(x0,ε)中最大的那个作为统一的δ(ε)会怎样？
好处：既然对最挑剔的点(需要最小δ的那个点)我都用了比他所需更大的δ，那对其它不挑剔的点，这个统一的δ肯定也够用

而一致连续性要求：存在一根“标准权杖”长度δ，使得全国任意两个相距小于δ的城镇，它们的法律执行情况都差不多(f(x)-f(x0)丨<ε )
这里没有固定的国王x0！权杖δ是提前统一制定的，它必须适用于任意两个地方（任意x1,x2）而不是围绕某个特定的国王。

任意ε>0：无论朝廷（数学定义）下达多么严苛的法律(ε)

存在δ>0：本王(x0)都能找到合适的统治半径(δ)

任意x:|x-x0|<δ：但凡是在我疆域的子民(x)

|f(x)-f(x0)|<ε：都自动遵守我的法律

那对于我们刚刚说过的统一的δ肯定也够用，那这个“最大的δ”真的存在吗？而δ(x0,ε)为什么可能没有最大值从而取了上确界呢？
因为δ(x0,ε)可能随着x0的变化可以无限接近某个数，但永远达不到。例如：对于f(x)=1/x在(0,1)上，当x0无限接近于0时需要的δ(x0,ε)可以无限小
但任何一个确定δ都不能成为“最大者”，因为总有更靠近0的点需要更小的δ
既然集合没有最大者那就上确界记为δ*(x0,ε)，可能属于δ(x0,ε)这个集合也可能不属于，属于就是最大值不属于就是上确界

但是取上确界δ*(x0,ε)作为统一的δ(ε)真的行得通吗？

用国王来理解就是：

参赛者：全国每个国王x0,x1,x2......

比赛项目：每个国王展示他能使用的最长安全权杖，记其长度为L(x0)

安全标准：权杖再长一点，他的疆域就会出现不法之徒
注意：这不是“谁需要多长的最小权杖”而是“谁最多能使用多长的权杖”

第二步：选拔过程

第一轮：各国内部选拔

在每个国王王国里，测量他能安全使用的最长权杖长度L(x0),L(x1),L(x2)......
这个L(x0)就是陈老说的δ*(x0,ε)（每个国王适用的δ的上确界）

第二轮：全国总决赛

把所有国王的L(x0)放在一起，即L(x0),L(x1),L(x2)......

找出这长度中的最短者（下确界）：Lmin = min{L(x0).L(x1),L(x2)...L(xn)}

第三步：为什么取最短的最长权杖？

让我们考虑两种极端情况

情况一（理想情况）

L(x0)=10m  L(x1)=8m  L(x2)=12m, 所以Lmin=8m

如果我们制定统一的权杖标准为7m，那各位国王都安全

情况二（有问题）

L(x0)=10m  L(x1)=0.01m  L(x2)=12m, 所以Lmin=0.01m

如果我们制定统一的权杖标准为0.005m,那各位国王也是安全的

但这个时候我们发现如果你的Lmin=0又或者无限趋近于0，因为你要求的统一的δ要>0啊，但现在找不到啊

如果Lmin=0,就意味着存在国王的安全半径可以无限小，你永远找不到一个统一的正数δ能覆盖所有人

这正是陈老说的：

inf x∈X δ*(x0,ε)>0, 这个δ*(x0,ε)就是我们的L(xn) inf就是我们的Lmin, 要求>0, 只有>0才能一致连续，=0不一致连续，这才是一致连续的充要条件

现在可以解释一下为什么绝大多数开区间连续但是不一致连续需要找反例了。

在国王模型中：

开区间意味着王国没有明确的边界，国王可以无限逼近边疆但永远不在边疆上。

而[a,b]有明确的边界国王a和b，最危险的国王一定在a，b或内部某个点，所以总能找到统一标准。
但(a,b)没有边界，最危险的国王可能出现在x0→a+或x0→b-，因为靠近边疆每个国王都很危险，没有最危险的只有更危险的，要讨论无穷种情况，只能构造反例。
